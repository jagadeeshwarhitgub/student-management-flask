# Student Management System (Flask)

A simple web application built using Python Flask and SQLite.

## Features
- Add student
- Edit student
- Delete student
- View student list

## Tech Used
- Python
- Flask
- SQLite
- HTML

## How to Run

1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
